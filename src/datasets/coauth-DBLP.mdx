This is a temporal higher-order network dataset, which here means a sequence of timestamped simplices where each simplex is a set of nodes.
In this dataset, nodes are authors and a simplex is a publication recorded on DBLP.
Timestamps are the year of publication.

<Alert type="warning">In the original dataset, some nodes are repeated within hyperedges. This version of the dataset has removed these duplicates.</Alert>

## Overview

The Cora co-citation network is a hypergraph where nodes are scientific papers from the Cora dataset and hyperedges are sets of papers that are co-cited together by another paper.
Each node is labeled with the paper's research field, chosen from seven computer science topics.
This dataset is commonly used as a benchmark in hypergraph learning tasks.

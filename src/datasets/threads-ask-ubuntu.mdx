## Overview

This temporal higher-order network represents discussion threads on Ask Ubuntu.
Nodes are users, and each hyperedge is the set of users participating in a thread.

> [!IMPORTANT]
> Singleton hyperedges, corresponding to threads with only one participating user, are omitted in this dataset.

## Attributes

- `thread_id : string`: The source thread identifier associated with a hyperedge.
- `time : datetime`: The thread timestamp in ISO 8601 format, relative to the normalized source epoch.

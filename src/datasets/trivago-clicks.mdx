This is a hypergraph, where nodes are accommodations (mostly hotels), and hyperedges are sets of accommodations for which a user performed the "click-out" action during the same browsing session, which means the user was forwarded to a partner site.
A few hyperedges are repeated.
The dataset was derived from the ACM RecSys Challenge 2019. Each node is labeled with the country where the accommodation is located.

## Usage

The dataset can be loaded directly in TopoNetX using the following code:

```python
import toponetx as tnx

nodes, hyperedges = tnx.datasets.load_benson_hyperedges("trivago-clicks")
```

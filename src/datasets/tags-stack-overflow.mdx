## Overview

This temporal higher-order network represents tag co-occurrence on Stack Overflow.
Nodes are tags, and each hyperedge is the set of tags applied to a question.

> [!IMPORTANT]
> Singleton hyperedges, corresponding to questions with only one tag, are omitted in this dataset.

## Attributes

- `name : string`: The tag name corresponding to a node.
- `post_id : string`: The Stack Exchange post identifier associated with a hyperedge.
- `time : datetime`: The question timestamp in ISO 8601 format.

# Tools

## Interacting with AHORN

If you want to programmatically interact with AHORN, we provide a JSON file that contains machine-readable data about all datasets in the repository. You can find it at <a href="https://ahorn.rwth-aachen.de/api/datasets.json" target="_blank" rel="noopener noreferrer">https://ahorn.rwth-aachen.de/api/datasets.json</a>.

For a more convenient experience, we also provide the Python package [`ahorn-loader`](/about/ahorn-loader) for easy access to the AHORN API.
